#MavisSite

Mavis Site foi feito para divulgar o novo album do artista undeground Mavis, mas a ideia é atualizar ele de forma regular toda vez que for lançada uma nova música.
O site foi feito por Girleno Gabriel e esse também é o primeiro trabalho publicado do próprio. As liguagens usada para o desenvolvimento do site foram somente Html5 e CSS3, além do
auxilio do Figma para se ter uma ideia prévia do design final. 
Ainda em desenvolvimento.
